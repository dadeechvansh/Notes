###### **Request Contains---**



1. Method - the action we want to perform.

&#x09;Examples: GET| POST | PUT | PATCH | DELETE

2\. URL - The address of the resource.

&#x09;Ex.- /students | students/101 | /courses

3\. Headers -  Extra information about the request.

&#x09;Ex. - Content-Type| Authorization Token | Accept Language

4\. Body -  Data Sent to the server.

&#x09;used mainly with POST | PUT | PATCH





###### **Response Contains---**



1. Status Code - Tells whether the request succeeded or failed
Example - 200 OK | 201 Created | 404 Not Found | 500 Server Error
2. Headers - Extra information about the response
Example- Content-type : application/json
3. Body - Actual data returned by the server

&#x09;Example Response

&#x09;{

&#x09;   "id":101, "name":"Akhil", "Course":"Python"

&#x09;}
